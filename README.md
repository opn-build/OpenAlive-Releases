# OpenAlive-Releases

Keep your Windows session active with minimal effort.

OpenAlive is a lightweight Windows utility designed to simulate occasional mouse and keyboard activity, helping prevent idle status and keeping your session active.

## Features

- Mouse activity simulation
- Keyboard activity simulation
- Configurable intervals
- Randomized behavior
- System tray support
- Lightweight resource usage
- Simple configuration
- No account required
- No cloud dependency

## Requirements

- Windows 10
- Windows 11

## Installation

Download the latest release from the Releases page and run:

OpenAliveSetup.exe

## Privacy

OpenAlive runs entirely on your local machine.

- No telemetry
- No data collection
- No cloud services
- No user tracking
